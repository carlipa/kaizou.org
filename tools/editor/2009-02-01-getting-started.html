--- layout: page title: Getting Started tags: [] status: publish type: page published: true meta: _edit_last: '1' _wp_page_template: default --- You don't need to install anything to start using the Kaizou editor. However, we must inform you that it works faster with the Kaizou Firefox extension.
  • Install Kaizou Firefox Extension.

Try Kaizou web online editor (beta)

Thanks to the Kaizou online editor, you will be able to modify any web page and see the result instantly. Two modification types are available:
  • Modify any block inner HTML,
  • Set an HTML node attributes,
Note: To save your modifications you will need the Firefox extension.

Browse Kaizou files on this site

For an overview of what is Kaizou, just browse this site and try the several sample files posted under the "Editor" category. Each file is presented with its description and two links:
  • the "See it in action" link will open the target page and apply transformations to it.
  • the "Download" link points to the actual XML Kaizou file.
